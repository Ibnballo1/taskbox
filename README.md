# 📖 [Task_App] <a name="about-project"></a>

**[Task_App]** is a straightforward and user-friendly task manager designed to help you stay organized and focused. With Taskbox, you can effortlessly add, delete, edit, and mark tasks as completed, making it the perfect tool to manage your daily to-dos. Whether you're a professional juggling multiple projects or simply aiming to boost your productivity, Taskbox is here to simplify your task management experience.
